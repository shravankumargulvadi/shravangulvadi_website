  • Objective: Utilize mobile robots like Unmanned Aerial Vehicles (UAVs) and Unmanned Ground Vehicles (UGVs) to prevent forest fires by removing potential fuel in forests.
    This project was done as part of CMU's SAFEFOREST initiative
  • Semantic SLAM Pipeline: Develop a Semantic Simultaneous Localization and Mapping (SLAM) pipeline for UAVs, integrating semantic segmentation and non-linear optimization
    for identifying fuel and accurately mapping forest environments.
  • Landmark Modeling and Robot Pose Estimation: Implement a semantic feature-based pose optimization to refine geometric models of landmarks (trees)
    using parameterized cylinder models fitted onto LIDAR point features, while concurrently estimating the robot's pose. Use a parameterized cylinder model
    fitted onto LIDAR points through Geometric Least Squares Optimization, enabling pose transformation estimation for robot motion. Accurate pose transformations
    between consecutive LIDAR sweeps were achieved using various optimization methods like Levenberg-Marquardt, Trust Region Reflective, BFGS, and Nelder-Mead.
  • Optimization Methods Comparison: Analyze different optimization methods, focusing on iterations, overall time, and real-time application
    suitability. Levenberg-Marquardt Method proved most efficient, averaging 0.09s per operation, aligning with a 10 Hz lidar frequency.
