  • Dense SLAM Implementation: Utilized point-based fusion for enhanced 3D localization and mapping.
  • Projective Iterative Closest Point for Visual Odometry: ICP is widely used for determining the visual odometry for tracking the camera movement,
    specifically projective ICP used in this project has two components:
    • Projective Data Association: Involved projecting a 3D point cloud onto a 2D plane, where projective nearest neighbors are identified and correlated.
    • Calculation of Rotation and Translation Matrices: Based on the data association from the projected points, rotation and translation matrices
      are computed using Iterative Closest Point (ICP) algorithm.
  • Mapping with Estimated Camera Pose: Projects points based on the estimated camera pose, followed by a filtering process to refine the data.
    Computes the weighted averages of various properties such as positions, normals, and colors to create a detailed and accurate map.
  • Implementation of Particle Filter/Monte Carlo Localization: Aimed to localize a lost mobile ground robot in an indoor environment.
    Implemented robot motion model, sensor model and ray tracing algorithm for successful 2D localization of the robot.
    • Prediction Step: Updating particle poses by sampling from the motion model.
    • Correction Step: Calculating an importance weight for each particle based on target and proposal distributions, essentially using the sensor model.
    • Resampling Step: Particles are resampled for the next time step based on their importance weights.
